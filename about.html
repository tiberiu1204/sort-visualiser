About Sort Visualiser

A sort visualiser is an interactive tool designed to demonstrate the process and efficiency of various sorting algorithms. By visually representing the steps each algorithm takes to sort a list of elements, typically numbers or bars of varying heights, users can better understand how these algorithms work. The visualisation often includes animations showing comparisons, swaps, and the progression of the list from unsorted to sorted order. This educational tool is invaluable for students, educators, and anyone interested in computer science, providing an intuitive grasp of concepts like algorithm efficiency, time complexity, and data manipulation.

Below, you can find pseudocode for all the algorithms implemented by the visualiser.

Bubble Sort Pseudocode

procedure BubbleSort(var A : array of Integer);
for i := 0 to length(A) - 1 do
for j := 0 to length(A) - i - 2 do
if A[j] > A[j + 1] then
swap(A[j], A[j + 1]);
end if
end for
end for
end procedure
